Com o grafo já sendo publicado no navigator gui
Na Interface do Navigator Gui altere o final goal da seguinte forma:
 (a) Insira um final goal próximo do ponto desejado.
 (b) Clique em 2D Map (fica abaixo de File)
 (b) Aperte a tecla seta para baixo (Isso é necessário para ativar o panel que reconhece o input das teclas)
 (c) Precione a tecla J para ativar a edição do final goal.
 (d) Use:
     - As setas , direita, para cima e para baixo
     - A letra L para mover para direita
     - A letra K para mover para esquerda     
Um arquivo chamado final_goal_pose.txt será gerado no bin com o ultimo final goal que você escolheu.
Obs.: Esse arquivo é modificado a medida que você anda nas poses.
